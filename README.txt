User poate vedea si cauta joburi si poate aplica la joburi. 
In Dashboard poate vedea istoricul la ce a aplicat.
User primeste email de confirmare dupa ce a aplicat la job 
si email daca cv-ul lui a fost vizualizat.
Angajatorul poate vedea cine a aplicat.
Angajatorul poate adauga firme, si in fiecare firma poate adauga joburi. 
Angajatorul poate CRUD la firmelele si joburile lui. Pentru joburi poate alege public sau draft status.


RUN CMD:
npm run dev
docker-composer up
